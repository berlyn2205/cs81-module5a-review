# Reflection

## What did you learn from reading and commenting on this code?

I learned how higher-order functions like map(), filter(), and reduce() can make JavaScript code shorter and easier to understand. I learned that map() transforms data into a new array, filter() selects items that match a condition, and reduce() combines values into a single result. Commenting on the code helped me understand how each function processes the hobby data.

## What was the most confusing part?

The most confusing part was understanding reduce() because it works differently than map() and filter(). It took some time to understand how the sum value changes during each loop and how the final total is returned.
